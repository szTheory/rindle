# Rindle

**Media, made durable.**

Phoenix/Ecto-native media lifecycle library. Rindle manages the full post-upload lifecycle: upload sessions, staged object verification, asset modeling, attachment associations, variant/derivative generation, background processing, secure delivery, observability, and day-2 operations.

## Installation

Add `rindle` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:rindle, "~> 0.1.0"}
  ]
end
```
