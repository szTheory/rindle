<objective>
Close the Phase 95 verifier gaps for mobile table behavior and stale screenshot documentation without
changing product scope or introducing dependencies.

Purpose: UPLIFT-01 requires the Level-1 table primitive to remain excellent across responsive states.
The current mobile rule stacks sticky tables because the CSS checks the modifier on the table while the
gallery marks the sticky wrapper. The same mobile stacked-table CSS consumes `td[data-label]`, but the
gallery emits unlabeled cells. The guide also documents only the original 10 screenshots while the
checker and ExUnit require 18.

Output: wrapper-aware generated CSS, non-empty data-labels on non-sticky gallery table cells, sub-760
browser assertions, updated 18-screenshot guide copy, regenerated/synced CSS artifacts, and
`95-05-SUMMARY.md`.
</objective>
