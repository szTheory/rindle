# 83-02: Derived assets

## Objective

Ship the raster-requiring surfaces from regenerable SVG sources: favicon set
(SVG + 16/32 PNG + .ico) and the 512px social avatar.

## Tasks

1. `favicon.svg` — simplified two-tributary cut (three become two below 32px),
   heavier strokes, square viewBox, dark-mode aware via `prefers-color-scheme`.
2. `social-avatar.svg` — mark (dark scheme) optically centered on a Deep Current
   512-square surface (the square is the avatar surface, not a mark container).
3. `render-derived.mjs` rasterizes exact-pixel `favicon-16/32.png` + `avatar-512.png`
   via Playwright; `favicon.ico` assembled from the PNGs with ImageMagick.

## Verification

- `node check.mjs` green (constraint + budget gates).
- PNGs verified real (16×16 RGBA, 32×32, 512×512 via `file`).
- 16px favicon legible in a mock browser-tab strip (screenshot evidence).
