## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| host runtime -> AV worker | Environment limits and disk pressure can make a valid recipe unsafe to run. |
| FFmpeg exit status -> durable upload | Exit 0 is not sufficient proof that the output is valid or complete. |
| temp run root -> maintenance cleanup | Crashes or timeouts can leave orphaned directories and child processes. |
