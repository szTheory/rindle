## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| helper opts -> rendered HTML | Caller-supplied attrs and poster values become rendered markup. |
| helper -> delivery surface | AV helpers must not bypass the delivery policy or invent URL semantics. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-27-01 | Tampering | `lib/rindle/html.ex` attrs/presenter path | mitigate | Continue to HTML-escape rendered values and reuse the existing attr pass-through helpers instead of hand-building new unsafe markup paths. |
| T-27-02 | Information Disclosure | `lib/rindle/html.ex` source fallback | mitigate | Skip non-ready variants and fall back to the original asset URL; never render stale or missing AV source URLs as if they were playable. |
| T-27-03 | Elevation of Privilege | `lib/rindle/html.ex` playback URL resolution | mitigate | Route all AV playback URL resolution through `Rindle.Delivery.streaming_url/3` rather than constructing URLs in the helper layer. |
</threat_model>
