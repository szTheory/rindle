# Plan 02-03: Image/Vix Processor Adapter

**Goal:** Implement the `Rindle.Processor.Vix` adapter using the `Vix` (libvips) library to handle image transformations based on profile recipes.

## Success Criteria
- [ ] `Rindle.Processor.Vix` implements the `Rindle.Processor` behaviour.
- [ ] Support for `:width`, `:height`, and `:mode` (crop, fit, fill) options.
- [ ] Support for `:format` (jpeg, png, webp) and `:quality` options.
- [ ] Processing returns a binary of the transformed image.
- [ ] Unit tests for `Vix` adapter using sample images.

## Proposed Changes

### Processor Adapter
- Create `lib/rindle/processor/vix.ex`.
- Implement `process(source, recipe, opts)` where `source` is a binary or path.

### Dependencies
- Ensure `vix` is in `mix.exs`.

### Testing
- Create `test/rindle/processor/vix_test.exs`.
- Test various resize modes and format conversions.

## Verification Plan

### Automated Tests
- Run `mix test test/rindle/processor/vix_test.exs`.
