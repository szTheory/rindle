phase's serialized ci.yml chain (wave 1); 106-03 (package-consumer split) depends on it landing first to

on (D-06).
Output: a top-level `concurrency:` block in ci.yml with branch-aware cancel-in-progress.
</objective>

| push:main run → release gate | `release.yml gate-ci-green` reads the push:main run conclusion; cancelling that run is a release-integrity hazard |
