## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| fork PR → CI | A fork PR is event_name == 'pull_request'; package-consumer-full is skipped, so no full-matrix secrets/cost on fork PRs |
| push:main run conclusion → release publish | release.yml gate-ci-green keys off this conclusion; the full matrix must genuinely fail the conclusion on any leg failure |
| labeled PR event → full lane | `pull_request: types:[...,labeled]` still yields event_name == 'pull_request' → full lane correctly stays skipped on a label |
