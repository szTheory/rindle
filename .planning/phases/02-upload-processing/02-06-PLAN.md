# Plan 02-06: LiveView Integration Helpers

**Goal:** Implement helpers for Phoenix LiveView to simplify the configuration and consumption of direct-to-storage uploads using Rindle.

## Success Criteria
- [x] `Rindle.LiveView.allow_upload/4` implemented to wrap Phoenix's `allow_upload`.
- [x] `Rindle.LiveView.consume_uploaded_entries/3` implemented to verify and promote direct uploads.
- [x] Integration documentation/examples for LiveView usage (moduledoc with usage example).
- [x] Unit tests for helpers (7 tests covering config, signer wiring, exports).

## Proposed Changes

### LiveView Helpers
- Create `lib/rindle/live_view.ex`.
- Implement `allow_upload(socket, name, profile, opts)`:
    - Sets up `:external` signing using `Rindle.initiate_upload/2` and `Rindle.presigned_put/4`.
- Implement `consume_uploaded_entries(socket, name, func)`:
    - Calls `Rindle.verify_upload/2` for each entry.
    - Transitions entries to `validating` and enqueues `PromoteAsset`.

### Testing
- Create `test/rindle/live_view_test.exs`.
- Mock socket and entries to test helper logic.

## Verification Plan

### Automated Tests
- Run `mix test test/rindle/live_view_test.exs`.
