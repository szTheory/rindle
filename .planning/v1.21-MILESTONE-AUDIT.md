gate"* — is achieved:

  both the merge-blocking `local` gate and `cover/excoveralls.json` from one `coveralls.multiple` run.
- **Subprocess `:epipe` race fixed** (109): `run_isolated/5` absorbs the MuonTrap #98 broken-pipe exit
  at the single AV chokepoint; the two originally-flaking tests pass unmodified; ships `fix:`→0.3.2.

  `Application.put_env(:rindle, :repo, …)` swap; `:global_repo_swap` guard rule makes it
  un-reintroducible; ships `fix:`→0.3.2.

  2026-06-26 cluster from silently regressing.
- **PR↔main gate gap closed LAST** (112): the lean `adoption-demo-e2e-smoke` lane joined
  `CI Summary.needs` only after the de-flake phases landed AND 3 consecutive green push:main runs were
  observed — the gate does not import a still-live flake.

The **load-bearing dependency order** (de-flake 108/109/110 → lock 111 → shift-left 112 LAST) holds in
the shipped artifacts: the smoke lane's needs-wiring commit (`68046c6`) is the last `ci.yml` touch,
